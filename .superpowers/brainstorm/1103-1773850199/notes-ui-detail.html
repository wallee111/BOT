Key Interactions

Auto-save
Debounced save (1s after last keystroke). Status indicator shows Saving → Saved.
New Note
Creates empty note in current folder, opens editor immediately with title focused.
Delete Note
Confirm dialog → removes from Firestore + cache. Returns to notes list.
Move to Folder
Dropdown of available folders. Updates folderId on the note.
Folder CRUD
New folder: inline text input. Rename: long-press/right-click. Delete: moves notes to All Notes.
Search
Client-side filter over cached notes. Matches title + content text. Results across all folders.
