Githu work flow


initialize as a new git repo (already done)
```
git init

```

add files to staging area when changes are ready to be commited. '-A' means everything

```
git add -A
```

commit changes as a with a commit message

```
git commit -m "Add a change to do something
```


Push changes to github:

1. Set remote track branch (already done)

```
git push origin -u  <branch name>
```


2. Normal push to remote branch
```
git push

```
